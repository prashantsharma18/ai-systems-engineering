# Courses

Annotated list of online courses. Notes reflect personal assessment of depth and quality.

---

## Math for ML

- **Mathematics for Machine Learning Specialization** — Imperial College London (Coursera). Covers linear algebra, multivariate calculus, and PCA. Good for building intuition with visual explanations.
- **Probability and Statistics** — Stanford Online (edX). Rigorous. Good complement to the above.

---

## Machine Learning

- **Machine Learning Specialization** — Andrew Ng / DeepLearning.AI (Coursera). Best introductory ML course. Intuition-first, well-paced.
- **fast.ai Practical Deep Learning for Coders**. Top-down, code-first approach. Teaches production thinking from the start. Free.

---

## Deep Learning

- **Deep Learning Specialization** — Andrew Ng / DeepLearning.AI (Coursera). 5-course series. Covers fundamentals through sequence models and transformers.
- **Stanford CS231n: CNNs for Visual Recognition**. Free lecture videos and notes. Best deep dive into CNNs and visual representations.

---

## LLMs and Generative AI

- **LLM Bootcamp** — Full Stack Deep Learning. Free. Covers LLM fundamentals, prompting, RAG, fine-tuning, and deployment end-to-end. Highly recommended.
- **Short Courses** — DeepLearning.AI. Multiple 1-2 hour courses on RAG, agents, fine-tuning, evaluations. Good for targeted skill gaps.

---

## MLOps

- **Machine Learning Engineering for Production (MLOps) Specialization** — DeepLearning.AI (Coursera). Covers the full ML lifecycle in production.
- **Made With ML** — Goku Mohandas. Free. Excellent MLOps course with clean code and modern tooling.

---

## MCP and Agentic AI

- **Anthropic documentation and cookbook**. Primary reference for MCP, Claude API, and Agent SDK.
