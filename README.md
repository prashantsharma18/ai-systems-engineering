## How I Learn

I learn by building and writing. For each topic I read the theory, implement it from scratch in a notebook, and then write a notes document that forces me to synthesize what I have understood. When something is worth sharing publicly, it becomes a blog post.

This means every folder in this repo has three layers: theory (README notes), code (notebooks), and occasionally a longer article (blogs). Projects are where I integrate concepts across layers and build something end-to-end.
