# Contributing

This is a personal learning repository. Pull requests are not expected.

If you spot an error in a notebook, a conceptual mistake in the notes, or a better way to approach something — feel free to open an issue using the [learning note template](.github/ISSUE_TEMPLATE/learning-note.md). Corrections and suggestions are genuinely welcome.

If you find this repo useful, feel free to fork it and adapt the structure for your own learning journey.
